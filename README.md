
# 🧠 AI Medicine Recommendation System

An intelligent Machine Learning-based web application that recommends suitable medicines based on user symptoms. This system helps users and healthcare professionals make informed decisions by analyzing symptoms and predicting possible treatments.

---

## 🚀 Live Demo
https://ai-medicine-recommendation-system-2.onrender.com/

---

## 📌 Features

* 💊 Medicine recommendation based on symptoms
* 🤖 Machine Learning powered predictions
* 📊 Data preprocessing & feature engineering
* 🧾 User-friendly interface
* ⚡ Fast and accurate suggestions
* 🔍 Scalable and customizable system

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Python (Flask)
* **Machine Learning:** Scikit-learn
* **Data Handling:** Pandas, NumPy
* **Visualization:** Matplotlib / Seaborn

---

## 🧠 How It Works

1. User inputs symptoms
2. Data is processed and cleaned
3. ML model analyzes patterns
4. System predicts possible disease
5. Recommends suitable medicines

👉 Such systems typically use patient data, symptoms, and ML models like decision trees, SVM, or neural networks to generate recommendations. ([GitHub][1])

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/33Ayushi/AI-Medicine-Recommendation-System.git
cd AI-Medicine-Recommendation-System
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000/
```
